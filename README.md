# Teste Prático Java

Projeto desenvolvido em Java com foco em:

- Programação orientada a objetos
- Manipulação de listas e mapas
- Cálculos com BigDecimal
- Separação de responsabilidades (Service e Report)

## Como executar

javac -d bin src/model/*.java src/service/*.java src/Main.java  
java -cp bin Main
